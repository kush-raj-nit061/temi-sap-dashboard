🔗 SAP BTP API Reference

Send this POST request from BTP to trigger a route programmatically:

POST http://192.168.4.27:8080/api/route Content-Type: application/json { "stops": [ { "location": "kitchen", "label": "Kitchen", "object": "Laptop", "waitForConfirm": true }, { "location": "entrance", "label": "Entrance", "object": "Monitor", "waitForConfirm": true } ] }

Robot returns SSE events to GET /events. Poll GET /status for current state.
